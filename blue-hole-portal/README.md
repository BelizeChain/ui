# Blue Hole Portal

Government administration dashboard for managing BelizeChain infrastructure.

## Features

- **Treasury Management**: Multi-signature treasury operations
- **Governance Dashboard**: Proposal monitoring and voting
- **Validator Network**: Real-time validator status and performance
- **System Monitoring**: Blockchain, AI, quantum, and storage metrics
- **Compliance Tools**: KYC/AML oversight and reporting
- **Activity Logs**: Complete audit trail of system events

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3002](http://localhost:3002)
